# VLC Stream Keeper  
Keeps a camera stream running in VLC.

## Description
This script is used to ensure an RTSP camera stream is displayed on a small
display driven by a raspberry pi 4, or similar device, with no attached
keyboard or mouse. It eliminates the need to remote into the desktop
GUI of the RPI4 and restart the stream ~1-2 days as observed by the author.  

## Dependencies:  
    python3-dbus  
    xdotool  
## Requires:
    python 3.x

## Usage
```bash
vlc_keepalive_space.py
```
